# Общая алгербра и теория чисел

Интерактивный конспект по алгоритмам теории чисел, реализованный в виде [Marimo](https://marimo.io/) ноутбука.

## Темы

1. Тестирование чисел на простоту. Решето Эратосфена
2. Разложение числа на простые множители (факторизация)
3. Алгоритм Евклида и расширенный алгоритм Евклида (НОД, НОК)
4. Операции по модулю. Быстрое возведение в степень
5. Решение системы линейных сравнений
6. Функция Эйлера
7. Вычисление остатка от деления через теорему Эйлера
8. Мультипликативное обратное по mod n
9. Китайская теорема об остатках

## Установка и запуск

### 1. Клонировать репозиторий

git clone https://github.com/squez/number-theory-algorithms.git
cd number-theory-algorithms

### 2. Создать виртуальное окружение

python -m venv .venv

Активация:

- Windows: .venv\Scripts\activate
- macOS / Linux: source .venv/bin/activate

### 3. Установить зависимости

pip install -r requirements.txt

### 4. Запустить ноутбук

marimo edit notebook.py --watch
